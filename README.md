# 🛒 Amazon Clone

This is my first frontend project built using HTML and CSS.

## 📌 About the Project

I created this project to practice HTML and CSS by recreating the layout of the Amazon homepage.

This project helped me understand:
- HTML page structure
- CSS styling
- Flexbox
- Navigation bars
- Hero section
- Product cards
- Footer design
